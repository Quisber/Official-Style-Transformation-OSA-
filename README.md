# Official Style Assistant (OSA) & StEva

Данный репозиторий содержит практическую часть производственной практики, а также выпускной квалификационной работы (ВКР), посвященной автоматизации стилистической адаптации текстов (перевод из публицистического стиля в официально-деловой) с использованием LLM и технологий PEFT.

## Описание проекта
Обучение было выполнено на базе модели **Llama 3.1 (8B)**, для работы использовалась уже квантованная версия от Unsloth - meta-llama-3.1-8b-bnb-4bit. 

**Ключевые особенности:**
* **Гибридный подход:** При выполнении проекта были использованы как общепризнанные ML методы, так и экспериментальные, разработанные с лингвистическим упором.
* **Стабильность модели:** Обучение происходило с триггер-словом, которое вшито внутри модели и исключает вероятность расхождения модели с изначальной задачей.
* **Уникальная метрика:** Помимо BERTScore создан интерфейс с метриками StEva, в основе ключевые характеристики стиля текста - сложность, отстраненность, эмоциональность и т.д.

## Структура репозитория
* `/data` — датасеты для обучения и валидации (в формате .jsonl). Также загружена полноценная csv-таблица, при желании можно загрузить в Google Sheets.
* `/engine` — папка с кодом для обучения модели.
* `/stylemetrics` — папка для стилеметрического измерителя StEva.
* `/NLPmetrics` — оценки по метрикам BERTScore.
* `app.py` — визуальный интерфейс обученной модели на базе Streamlit.
* `determine_parameters.py` — код для определения оптимальных настроек LoRA
* `training.py` — непосредственно код для обучения и с подставленными гиперпараметрами.
* `stylemetry_evaluator` — "мозг" оценки по лингвистическим характеристикам.
* `stylemetry_visual` — интерфейс StEva.
* `requirements.txt` — список необходимых библиотек и их версий.



## Установка и запуск

1. **Клонирование репозитория:**
   ```bash
   git clone https://github.com/Quisber/TST-TransformersBased.git

2. **Установка библиотек и зависимостей (Python 3.11)**
   ```bash
   pip install -r requirements.txt

3. **Скачивание адаптеров**
   ```bash
   pip install huggingface_hub
   huggingface-cli download Quisber/Llama-3.1-7b-Official-Style-Lora --local-dir checkpoints/checkpoint-500

4. **Запуск интерфейса модели**
   ```bash
   streamlit run engine/app.py

5. **Запуск StEva интерфейса (Optional)**
    ```bash
    streamlit run stylemetrics/stylemetry_visual.py
